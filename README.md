# task-clean
> Clean files and folders.

## The "clean" task

### Usage Examples

```js
var clean = new (require('task-clean'))
clean.run(inputs, options, logger)
```

### Options

#### options.force
Type: `boolean`
Default: 'false'

If delete CWD or outside CWD enabled.
